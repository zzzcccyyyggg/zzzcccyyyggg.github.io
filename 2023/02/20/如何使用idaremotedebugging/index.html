如何使用IDA Remote Debugging

发表于2023-02-20|更新于2024-01-13
|字数总计:267|阅读时长:1分钟|阅读量:

1、设置允许Dbug

在kali linux.vmx文件中加入下面三行(这里各自的文件名可能不太相同 ) debugStub.listen.guest64 = "TRUE" debugStub.hideBreakpoints= "TRUE" debugStub.listen.guest64.remote = "TRUE"

这一步好像不一定需要 笔者最后是用Ubuntu实现的 但是并没有添加以上三行

2、配置Ubuntu

在ida安装的文件夹处找到linux_server64与linux_server（分别用于调试64位与32位）并移动到linux中（自己选个文件夹 调试的文件与以上两个文件需要在同一个文件夹中）

记得用chmod 777 文件名将文件名的权限都进行修改

3、ida配置

首先查看虚拟机的ip （可用ifconfig指令查看）

在IDA菜单中，选择“Debugger->Process options”，打开要调试的程序设置选项，在Application和Input file两项中输入kali中要调试的程序，在Directory中输入要调试的程序在kali中的目录，在Hostname中输入linux机器的IP地址，端口保持默认即可，密码写虚拟机设置的密码

然后就完成啦
